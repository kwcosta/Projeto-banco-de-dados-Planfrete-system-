## 📌 Sobre o Projeto
O **Planfrete System** é um sistema de gestão projetado para centralizar, organizar e conferir total visibilidade ao fluxo de fretes de uma empresa, mitigando erros operacionais e integrando nativamente o setor de Logística ao de Contabilidade.

---

## 📂 Estrutura do Repositório

Aqui você encontra todos os artefatos entregáveis do projeto:

*   **`minimundo.pdf`**: Contextualização, descrição das regras de negócio e perspectivas de evolução do sistema.
*   **`modelo_logico.png`**: Diagrama do modelo lógico/relacional utilizando a notação Pé de Galinha.
*   **`modelo_fisico.png`**: Diagrama do modelo físico explicitando os tipos de dados e chaves (PK/FK).
*   **`schema.sql`**: Script de criação das tabelas e restrições do banco de dados no MySQL.
*   **`data.sql`**: Script de carga de dados iniciais e exemplo de transação segura.
*   **`queries.sql`**: Scripts contendo as 10 consultas operacionais e analíticas solicitadas.
